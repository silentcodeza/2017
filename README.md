## 🚀 Getting Started

### Prerequisites
- **Java JDK 8+**
- IDE: **IntelliJ IDEA**, **Eclipse**, or **NetBeans**
- Or command line with `javac` and `java`

### Running Programs

```bash
# Clone the repository
git clone https://github.com/silentcodeza/2017.git
cd 2017

# Navigate to source code
cd "Source Code"

# Compile a Java file
javac FileName.java

# Run the compiled class
java FileName
```
